
# Password Generator

A simple password generator app written in c++.

## Dependencies

1. cmake
2. make
3. C++11 compiler

  
## Installation 

1. Clone the repo.
```bash 
git clone https://github.com/Abir10101/password_generator.git
```
2. Make build folder
```bash 
cd password_generator
sh ./build.sh
sh ./run.sh
```
And thats it. Enjoy!
