
# Password Generator

A simple password generator app written in c++.

## Dependencies

1. cmake
2. make
3. C++11 compiler

  
## Installation 

1. Clone the repo.
```bash 
git clone https://github.com/Abir10101/password_generator.git
```
2. Make build folder
```bash 
mkdir build && cd build
```
3. Compile the program
```bash 
cmake -S .. -B . && make
```
4. Run
```bash 
./password_generator
```
Enjoy!
